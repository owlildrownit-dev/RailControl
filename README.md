# RailControl

Автономная информационная система управления железнодорожным вокзалом для учебной практики МТУСИ.

Студент: Кривошеин Даниил Романович  
Группа: УБВТ2401

## Запуск

Распаковать архив и открыть `index.html` в современном браузере. Приложение не требует установки зависимостей, сервера или подключения к интернету.

## Структура

- `index.html` — точка входа;
- `assets/css/styles.css` — оформление интерфейса;
- `assets/js/storage.js` — конфигурация, демонстрационные данные и локальное хранилище;
- `assets/js/local-api.js` — клиент локального API с операциями чтения, создания, изменения и удаления;
- `assets/js/app.js` — интерфейс, навигация, формы и бизнес-логика;

## Хранение данных

Данные сохраняются в `localStorage`, а текущий вход — в `sessionStorage`. Локальный API имитирует асинхронные обращения к серверу. При переходе на серверную архитектуру объект `api` можно заменить HTTP-клиентом, сохранив интерфейс приложения.
